HSN Code Mismatches

HSN (Harmonised System of Nomenclature) codes determine GST tax brackets. When a vendor files the wrong HSN code for an item, it lands in a higher tax bracket — and the company pays the difference. This is sophisticated tax manipulation: the item name and unit price are correct, only the classification changes.
